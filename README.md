# ToDo App
- group : Classify todo.
- todo : Create To Do for Users.

## Server setting
### install
`npm install`
### run
`node app`

## Database setting
### 1.Server connect
`mysql -uroot -p`
### 2.Create and Choose DB
`mysql> create database todo_db default character set utf8 collate utf8_general_ci;`
`mysql> use todo_db;`
